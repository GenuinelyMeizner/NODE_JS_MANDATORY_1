Node.js

What is Node.js?

Node.js is a open-source event driven server-side development environment, that offers cross-platform compatibility and fast code execution.

It utilizes the Javascript V8 engine which offers fast code execution and enables full front-end/back-end control through only Javascript.

Furthermore, it is run on a single thread with event looping which offers non-blocking user requests and no buffering of data. It also acts asynchronously thus it never waits for an API to return data.

Node.js is released under the MIT license, so it is completely free to use.

Why use Node.js?

Javascript V8 Engine

The Javascript V8 engine offers back-end development through Javascript and is generally considered to have very fast code execution compared to traditional server-side/networking applications.

Cross-platform compatibility

You are able to deploy all Node.js projects to nearly every operating systems in use, although between UNIX based operating systems and Windows you may have to make alterations to your code, specifically your paths as UNIX and Windows use different indexing methods.

Asynchronous and event driven

Every API you make is asynchronous, which means that your server never waits for the API to return data, but just moves along to the next call. A notification event within Node.js will notify the server to get the requested data from the previous API call.

Single Threaded

Node.js uses just a single thread to provide service to it's users, through it's event looping it makes the Node.js server much more scalable than traditional servers.

No-Buffering

The server never buffers any data, but just outputs data.

MIT License

Node.js is completly free to use, both commercially and privately.

Download Node.js through https://nodejs.org/en/.
