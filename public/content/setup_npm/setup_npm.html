npm

What is npm?

Node Package Manager is the inbuilt package manager that sorts out all of your dependencies.

Your project dependencies are all the community/commercially made libraries/frameworks that you want to add to your project.

How do you use npm?

My opinion of a standard Node.js project would need the following dependencies.

  • Express
  • Nodemon
  • Bootstrap

There are two ways (maybe more) to install npm's, both ways require us to open a terminal.

Predetermined package.json

We can predetermine which npm's we want to install and then install them all in one go. To do this we need to create a file called package.json, it should be placed on the top-level of your projects directory. In the package.json file you can manually write all the different libraries that you want to depend on.

Disregard everything in the picture except the package.json file placement and content.

When you've written all the different dependencies you want, you need to run a command through the terminal.

Using the command 'npm install' will install all the dependencies that you've written in the package.json file.

Manual npm install

If you want to install each library one by one, you still need a package.json file and a terminal open, but you do not need to fill out the dependencies, as the installation will do that for you automatically.

Instead of writing all the dependencies you just install the specific npm through a command.
